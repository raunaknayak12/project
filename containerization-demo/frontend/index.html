🐳 Container Task Manager

Running inside Docker — Frontend · Backend · Database

    How This Works

    🖥️ Frontend

    Served by Nginx container on localhost:3000. Pure HTML, CSS, JS.

    ⚙️ Backend

    Node.js + Express container on port 5000. Handles REST API.

    🗄️ Database

    PostgreSQL container on port 5432. Stores tasks persistently.
